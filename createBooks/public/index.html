New project in the makin... BOOKS!
a monolit application, a serverside rendering app where everything is created and rendered in the express server.

We 're going to:

  • create a library system
  • be able to list all the books in the library
  • show information about a specific book
  • be able to lend a book
  • be able to add a new book

Some more specifications:

  • how to lend a book is through a get route
  • how to add a book is through a post
  • and books could be from a global object array i books.js.
and more,
on the root page for the Book router:
  • every book should be shown
  • Show name on book and rented or available
  • and there should be a link to page "ADD NEW BOOK"
book information page should include:
  • name of the book
  • author
  • pages of the book
  • available or rented
  • and there should be a link/CTA to lend the book!
On the page: ADD NEW BOOK:
  • should include a form
  • the form should be able to add new books to the library
